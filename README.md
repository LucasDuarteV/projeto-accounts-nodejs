# 💰 Accounts

Projeto desenvolvido em Node.js durante o curso **Node.js: Express, MongoDB, MySQL, React.js e Arquitetura MVC**.

## 📖 Sobre o projeto

O Accounts é uma aplicação de terminal que simula operações bancárias básicas, permitindo ao usuário criar contas, consultar saldo, realizar depósitos e efetuar saques.

O objetivo do projeto é praticar conceitos fundamentais do Node.js, como:

* Manipulação de arquivos
* Entrada e saída de dados no terminal
* Estruturas condicionais
* Funções
* Modularização
* Uso de bibliotecas externas com npm

## 🚀 Funcionalidades

* ✅ Criar conta bancária
* ✅ Depositar valores
* ✅ Consultar saldo
* ✅ Sacar valores
* ✅ Encerrar o programa
* ✅ Persistência de dados em arquivos

## 🛠️ Tecnologias utilizadas

## ⚙️ Instalação

Clone o repositório:

```bash
git clone <url-do-repositorio>
```

Acesse a pasta do projeto:

```bash
cd accounts
```

Instale as dependências:

```bash
npm install
```

## ▶️ Executando o projeto

```bash
node index.js
```
