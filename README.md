# 💰 Projeto Accounts

Sistema bancário simples desenvolvido em Node.js durante os estudos de Backend.

## 📋 Sobre o Projeto

O Accounts é uma aplicação de linha de comando (CLI) que simula operações bancárias básicas, permitindo ao usuário criar contas, realizar depósitos, consultar saldo e efetuar saques.

O projeto foi desenvolvido com foco na prática dos conceitos fundamentais do Node.js, manipulação de arquivos e interação com o terminal.

## 🚀 Funcionalidades

* Criar conta bancária
* Realizar depósitos
* Consultar saldo
* Efetuar saques
* Validar operações
* Encerrar o sistema
* Persistência de dados utilizando arquivos JSON

## 🛠 Tecnologias Utilizadas

* Node.js
* JavaScript
* npm
* Inquirer
* Chalk
* File System (fs)

## ⚙️ Como Executar

### Clone o repositório

```bash
git clone https://github.com/LucasDuarteV/projeto-accounts-nodejs.git
```

### Acesse a pasta

```bash
cd projeto-accounts-nodejs
```

### Instale as dependências

```bash
npm install
```

### Execute o projeto

```bash
node index.js
```

## 🎯 Operações Disponíveis

### Criar Conta

Permite criar uma nova conta bancária.

### Depositar

Adiciona saldo à conta informada.

### Consultar Saldo

Exibe o saldo atual da conta.

### Sacar

Realiza saques respeitando o saldo disponível.

### Sair

Encerra a execução do programa.
